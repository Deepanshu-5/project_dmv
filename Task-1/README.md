# Scientific Calculator

A command-line based Scientific Calculator built using Python.

## Features

* Basic arithmetic operations
* Trigonometric functions
* Square root and logarithm
* Degree and radian conversion
* `pi` support
* Exception handling

---

## Technologies Used

* Python
* `math` module
* Lambda functions
* Dictionary mapping

---
